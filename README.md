### Image distribution

* test : `docker pull donghunc/tray:test`
* main : `docker pull donghunc/tray:latest`
* dev : `docker pull donghunc/tray:dev`

test는 main에  용도로 사용. 작동은 할거임
dev는 개발용으로 사용. 테스트 전 단계. 작동하는지 안하는지 모름
main은 작동하는 지금 운영 버전
